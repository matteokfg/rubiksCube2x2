# rubiksCube2x2

Simulador 3D de cubo magico 2x2 feito com Pygame e PyOpenGL.

## Controles

- `U`, `D`, `R`, `L`, `F`, `B`: gira uma face.
- `Shift` + movimento: gira a face no sentido inverso.
- Mouse esquerdo + arrastar: gira a camera.
- Scroll do mouse: aproxima ou afasta.
- Setas: ajuste fino da camera.
- `Espaco`: embaralha.
- `Tab`: desfaz o ultimo movimento.
- `Backspace`: reinicia.
- `Esc`: fecha.

## Teste rapido das mecanicas

```bash
python rubiks_cube.py --test
```
